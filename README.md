# knowledge-media-ai
An AI-powered editorial team that inspires curiosity and lifelong learning.

# 🌍 Project Polaris

> **知識は、分断ではなく、好奇心を生み出すものである。**

AIの力で、人々の知的好奇心を育て、学びや挑戦のきっかけとなるコンテンツを届けるオープンプロジェクトです。

---

# このプロジェクトについて

インターネットには、毎日膨大な情報が生まれています。

その一方で、注目を集めるために恐怖や怒りを煽る情報、真偽の分からない情報、誰かを傷つけるコンテンツも少なくありません。

私たちは、その流れとは異なるメディアを目指します。

Project Polaris が届けたいのは、「情報」ではなく「好奇心」です。

動画を見終わったあとに、

* 「世界って面白い。」
* 「こんなことがあるんだ。」
* 「誰かに話したい。」
* 「もっと調べてみたい。」
* 「自分も挑戦してみよう。」

そんな気持ちが自然と生まれるようなコンテンツを、AIと人が協力して作り上げます。

---

# Mission

世界中の知識を、誰もが楽しめる形で届ける。

私たちは単に情報を届けるのではありません。

人々の知的好奇心を育て、

学び、

挑戦、

そして共有したくなる発見を届けます。

---

# Vision

世界中の人が、

「世界って面白い。」

と毎日思える社会をつくること。

年齢や国籍、職業に関係なく、

誰もが学び続け、

新しいことへ挑戦できる世界を目指します。

AIは、人を置き換える存在ではなく、

人の可能性を広げるパートナーであると私たちは考えています。

---

# 私たちが大切にすること

私たちは常に、

* 好奇心を育てること
* 人を尊重すること
* 正確な情報を届けること
* 学びにつながること
* 希望を届けること

を大切にします。

そして、次のことは決して行いません。

* 個人への攻撃
* 誹謗中傷
* 陰謀論の拡散
* デマや誤情報の発信
* 恐怖や怒りだけを利用したコンテンツ制作
* クリック数だけを目的とした誇張表現

---

# AI編集部

Project Polaris は、一つのAIではありません。

役割を持った複数のAIが、一つの編集部として協力します。

| AI      | 役割              |
| ------- | --------------- |
| Polaris | 編集長（テーマ選定・意思決定） |
| Orion   | リサーチャー（情報収集・調査） |
| Athena  | ライター（台本・記事作成）   |
| Sophia  | 校正・ファクトチェック     |
| Atlas   | 動画制作・配信         |

それぞれが専門性を持ち、信頼できるコンテンツを生み出します。
