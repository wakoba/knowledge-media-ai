# Atlas Video Plan

Date: 2026-07-04

## Title

見えない手でつかむ：音で物体が浮く理由と空中触覚のリアル

## Video Concept

超音波フェーズドアレイで空中に音圧の山と谷＝音響ホログラムを作り、ビーズや液滴を非接触で捕捉・移動する仕組みと、その原理で空中に触感や図形を描く最新のヒューマンインタフェースを、物語としてわかりやすく紹介する。

## Estimated Duration

10 minutes

## Visual Style

クリーンな机上スケールの世界観。白背景やダークグレー背景に、青〜シアン系の音波フィールドを半透明の等値面で可視化。シンプルなモーショングラフィックスと3D図解を中心に、抽象表現（山と谷、渦、点群）の比喩を多用。最小限の実写（手元・指先・液滴）に安全な合成を重ねる。

## Target Viewer

テクノロジーに興味のある一般視聴者、HCIやVRに関心のある学生・エンジニア、理科好きの中高生

## Opening Visual

机の上に淡い波紋だけが見える暗室風のショット。小さなビーズが空中でピタッと止まり、指先が近づくとビーズがわずかに位置を保ったまま、指には“押し返し”のイメージが光の点で返る。直後にタイトルが音の等値面から浮かび上がる。

## Scene Plan

### Scene 1: Hook：見えない手が現れる

Narration Summary:

机の上に見えない手があって、ビーズが宙で止まり、指先には押し返される感覚が返ってくる。しかも全部、音だけで、という驚きを提示する。

Visual Description:

暗めの背景に半透明の青い波面。浮いたビーズが静止。指先に小さな光点が触れ、わずかな“たわみ”のエフェクト。最後に「え、ほんとに？」のテキストがフェードイン。

On-screen Text:

見えない手で、つかむ。音だけで、ほんとに？

Asset Type:

実写（指先）＋CG合成（音場可視化）

Editing Notes:

音の“パルス”に合わせてクイックカット→一拍置いてタイトルへ。低域のブーンではなく軽いパルス系SEで期待感を演出。

Safety Notes:

実写は低出力・短時間の演出とし、耳元や顔面至近での長時間照射は避ける旨を撮影前に周知。ペットは撮影エリアに入れない。

### Scene 2: イントロ：今日の旅の地図

Narration Summary:

音響浮揚と空中触覚ディスプレイの最前線を、超音波・ホログラム・見えない手というキーワードで探ることを宣言。

Visual Description:

机上サイズの装置を連想させるシルエット。キーワード『超音波』『ホログラム』『見えない手』が音の等値面から立ち上がるタイトル演出。

On-screen Text:

音響浮揚 × 空中触覚ディスプレイ｜超音波・ホログラム・見えない手

Asset Type:

モーショングラフィックス

Editing Notes:

タイトルは短く留めて本編へスムーズに移行。色はシアン基調で統一。

Safety Notes:

以降のデモは再現CGまたは安全出力の実写合成で表現する旨を小さく明示。

### Scene 3: なぜ今、音で“つかむ”のか？

Narration Summary:

非接触の価値（汚れない・壊れにくい・衛生的）。研究現場や日常UIでの利点。物体操作・表示・触覚が同じ“音の手”でつながる背景。

Visual Description:

白手袋とピンセットでの接触操作→同構図で空中で液滴が移動する非接触にマッチカット。公共端末の空中ボタンに指が近づくと淡い波紋が広がるアニメ。

On-screen Text:

非接触＝汚れない・壊れにくい・衛生的｜“物体操作・表示・触覚”が音でつながる

Asset Type:

CGアニメーション＋再現Bロール風合成

Editing Notes:

接触→非接触の切替はマッチカットで“違い”を強調。波紋エフェクトは弱めで上品に。

### Scene 4: 音の山と谷をデザインする

Narration Summary:

多数の超音波スピーカーの位相をずらして干渉を作り、狙った位置に音圧の“点・面・渦”を出現させる。音響ホログラム、波長と焦点サイズ、周波数と減衰のトレードオフ。

Visual Description:

小型トランスデューサの格子から出る波が重なり、空中に明暗の干渉パターンが発生。定規と波のアニメで波長と焦点サイズの関係を直感表示。周波数を上げると焦点が縮み、同時に減衰が増える矢印の対比。

On-screen Text:

位相制御→干渉→音響ホログラム｜40 kHz→波長 約8.6 mm｜焦点サイズ ≈ λ/2｜周波数↑ 焦点↓ 減衰↑

Asset Type:

3D図解アニメーション

Editing Notes:

干渉パターンの明滅は視認しやすい速度に。テロップは用語→一言要約の順で出す。

### Scene 5: どうして浮くの？ 小さな力の積み重ね

Narration Summary:

音響放射圧により粒子や液滴が圧力勾配に沿って安定位置へ。対象により節か反節に落ち着く。Gor'kovポテンシャル、力のスケール、単面アレイでのツイントラップや渦ビーム、対向配置の安定性。

Visual Description:

ボウル状のエネルギー地形にビーズが転がり“谷”にハマるCG。単面アレイの渦（ボトル）ビームで周囲が壁状になり中央に閉じ込められる様子。対向2面で安定度が上がる比較図。

On-screen Text:

音響放射圧｜Gor'kovポテンシャル｜力：数マイクロニュートン｜ミリメートル級を捕捉｜単面：渦・ツイントラップ／対向：より強固

Asset Type:

物理概念図解CG

Editing Notes:

“節/反節”は簡易アイコン（静かな点/大きく揺れる点）で示す。用語は出しっぱなしにせず要点で再提示。

### Scene 6: 動かす・描く・同時にこなす

Narration Summary:

焦点はミリ秒以下で書き換え可能。安定点を動かせば追従移動。高速走査で立体の輪郭を“描く”。振幅を100〜300 Hzで変調すると空中で押される触感。数kHz〜数十kHzの変調で指向性の可聴音も生成可能。

Visual Description:

単一ビーズが走査され、空中に蝶の輪郭がライトトレースで見える。指先上に光点が100〜300 Hz風のリズムで明滅し、指腹がわずかにたわむマクロ演出。可聴音シーンは細い音のビームが前方に伸びるCG。

On-screen Text:

焦点：ミリ秒以下で書換｜100〜300 Hz 振幅変調→“押される”触感｜数kHz〜数十kHz 変調→指向性の可聴音

Asset Type:

CGアニメーション＋実写合成

Editing Notes:

走査の残像は残しすぎず“描いている感”を優先。可聴音の表現は控えめなビームとクリックSEで伝える。

Safety Notes:

装置表現は机上スケール・安全出力の想定。長時間の至近距離曝露を避け、ペットや小動物への配慮が必要である旨を画面端に注意表示。

### Scene 7: 多点を作るコツ：ホログラム計算と時間多重

Narration Summary:

複数触点やトラップを作るには位相分布を解くホログラム計算（Gerchberg–Saxton系など）や時間多重を用いる。対象が増えるほど一つあたりの力は低下。カメラやマイクによる閉ループ制御で安定化。

Visual Description:

点群が高速でパチパチ切替しながら“面”に見えるアニメ。最適化前は点の明るさが不均一→後に均一化するヒートマップ。カメラとマイクのアイコンが制御ループで装置に戻る図。

On-screen Text:

ホログラム計算（Gerchberg–Saxton系など）｜時間多重で“同時”に見せる｜対象↑→1点の力↓｜閉ループ制御で安定化

Asset Type:

図解アニメーション

Editing Notes:

ヒートマップは色覚多様性に配慮した配色に。点滅は過剰にならないようフリッカー注意。

### Scene 8: 見える・聴こえる・触れるを一台で

Narration Summary:

単一ビーズで立体文字や蝶の輪郭を描く。液滴の空中合流と混色。見えないボタンの触感と微小な渦の視覚ヒント。押した場所だけにクリック音が届く。発泡スチロール片を運ぶ迷路チャレンジ。

Visual Description:

ライトトレースで“文字/蝶”。マクロで赤と青の液滴が宙で合流し境界が消える。手のひら上に光る空中ボタン、押すと局所に波紋とクリック音のビーム。俯瞰の迷路を粒子がたどるゲーム風演出。

On-screen Text:

一台三役：見る・聴く・触る｜空中で“描く” “混ぜる” “押す” “運ぶ”

Asset Type:

実演再現CG＋モーショングラフィックス

Editing Notes:

各例は短くテンポよく。クリック音はワンポイントで小気味よく配置。

Safety Notes:

液滴表現は安全なモデル液体で再現。手元デモは短時間・安全出力を想定。

### Scene 9: ありがちな誤解とその正体

Narration Summary:

磁石や静電気ではなく機械的な音圧。単面でも三次元捕捉は可能（対向はより強固）。対象は今はミリメートル級が中心。超音波は適切設計で安全に運用できるが配慮点あり。光学トラップとは特性が異なる。

Visual Description:

磁石アイコンに×、音の矢印が粒子を押す図。単面アレイと対向アレイの比較図。サイズバーで“今はミリメートル級”をハイライト。安全の配慮アイコン（出力・時間・ペット・副音）。光学トラップとの違いを二列比較の抽象図。

On-screen Text:

磁石/静電気ではない→音圧｜単面でも捕捉可（対向はより強固）｜対象：いまはミリメートル級｜安全＝出力・時間・ペット・副音に配慮｜光学とは特性が異なる

Asset Type:

図解モーショングラフィックス

Editing Notes:

否定表現は穏やかに。赤い×は小さく、代替の正解図を主体に。

Safety Notes:

“適切な出力管理・遮音・運用時間”に触れるテロップを明示。ペット同室での稼働は避ける旨を補足。

### Scene 10: 社会と未来への意味

Narration Summary:

非接触で持つ・運ぶ・混ぜることで汚染リスクや不純物を抑え、洗浄コスト低減の可能性。微小サンプルやバイオに適性。空中触覚は衛生的で壊れにくいUIとして車載・医療・公共空間で有用。視覚・触覚・音の一体提示はAR/VRやアクセシビリティ向上にも寄与。机上スケールの実装性も魅力。

Visual Description:

ラボの液滴操作→工場ライン→博物館展示へと繋がるモンタージュ。AR/VRユーザーが空中パネルを押して反応するシーン。

On-screen Text:

非接触で“持つ・運ぶ・混ぜる”｜衛生・コスト低減の可能性｜車載・医療・公共UI｜AR/VR・アクセシビリティ｜机上スケールで実装しやすい

Asset Type:

モンタージュCG＋再現Bロール風合成

Editing Notes:

シーン間は同じ音波モチーフでワイプ。希望を感じるBGMに移行。

### Scene 11: スケールアップの物理と安全設計

Narration Summary:

周波数を上げれば焦点は小さくなるが空気中では減衰が増える。放射力はエネルギー密度と体積におおむね比例、多数同時制御で1点あたりの力は下がる。閉ループ制御が助けに。安全面ではペットや可聴副音、出力と遮音の配慮が必要。

Visual Description:

“焦点縮小↔減衰増大”のシーソー図。点数を増やすと一点の力が下がるバー図。カメラ/マイクの閉ループブロック図。筐体断面に吸音材、ペット配慮アイコン。

On-screen Text:

周波数↑ 焦点↓ 減衰↑｜同時点数↑→1点の力↓｜閉ループで安定化｜出力管理・遮音・ペット配慮

Asset Type:

図解アニメーション

Editing Notes:

要点を大きく表示し、字幕で補足。安全アイコンは穏やかな色調で。

Safety Notes:

長時間・高出力は避ける設計と運用。可聴の副音への配慮。稼働時はペットを離す。

### Scene 12: クロージング：あなたなら何をつかむ？

Narration Summary:

見えない手が机上にある未来。何をつかむかを問いかけ、最適なバランス探索の旅が続くこと、世界が音でまだまだ面白くなることを締めくくる。

Visual Description:

冒頭の机上カットに戻り、波の等値面がゆっくり消えながら余韻。最後に小さなビーズが点となってフェードアウト。

On-screen Text:

あなたなら、まず何をつかむ？｜世界は、音でまだまだ面白くなる。

Asset Type:

モーショングラフィックス＋実写合成

Editing Notes:

BGMをフェードダウン、環境音を少しだけ残してエンドカードへ。

## Subtitle Direction

読みやすい白字＋半透明黒帯。1行12〜16字・最大2行を目安に短いセンテンスで表示。専門用語は最初に出た際に括弧で短く補足（例：位相＝発振タイミング）。重要な数値やレンジは色で軽く強調。点滅や過度な動きは避け、フリッカーに配慮。安全に関する注意は画面下に常時は出さず、該当カットで明確に提示。

## Narration Direction

落ち着いたトーンで、驚きを抑えた好奇心を前面に。専門語→一呼吸→平易な言い換えの順で説明。比喩（山と谷・ボウル・渦）を声でも繰り返し、視覚メタファーと同期。テンポは聞き取りやすさ優先で、小項目間に短い間を置く。誇張表現は避け、“いまは〜が中心”など現実的な言い回しを大切に。

## Thumbnail Direction

案A：宙に浮く水滴の接写＋背後にぼんやり格子アレイ。水滴の周囲に淡い波紋。テキスト『見えない手でつかむ』を上段に小さく、『音で浮かす？』を下段に太字。案B：指先の数センチ上に発光する“押せるけど触れない”ボタン。周囲に最小限の波紋エフェクト。テキスト『空中に“押せる”感覚』。案C：音の渦で小さなビーズが中央に留まるCG。渦の壁を可視化。テキスト『音のホログラム』。

## Production Notes

- 台本の事実・数値のみを使用。新規の数値・主張は追加しない。
- 著作権・ブランド固有素材に依存せず、すべてオリジナルの図解・CG・再現カットで構成。
- 音の表現はSEで演出するが、超音波自体は無音である点をナレーションで誤解なく扱う（台本の範囲内で）。
- 目に優しい配色（高コントラストすぎないシアン系）と色覚多様性への配慮を行う。
- 点滅・フリッカー・高コントラストの急変は避け、視認性と安全性を重視。
- 手元や指の実写は短時間・安全出力の想定で撮影。顔面や耳への近接カットは避ける。
- ペットや小動物は収録環境から退避。安全注意テロップを該当シーンで明示。
- BGMは希望と探究心を感じるミニマルなテクスチャ。クリックなどのUI音は控えめで明瞭に配置。
- 字幕・テロップは用語→要約→補足の順に短く。画面の混雑を避ける。
- セクション間のトランジションに“等値面”モチーフを統一的に用い、世界観を一貫させる。
