# Editorial Meeting

Date: 2026-07-04

## Topics

### 1. ひびを“自分で塞ぐ”コンクリート：マイクロカプセルが都市インフラを長寿命化

- Score: 92
- Summary: 微細なひびが入った瞬間にカプセルが破れて樹脂が流出・硬化し、ひびを自動で充填する“自己治癒コンクリート”。橋やトンネル、建物の耐久性を高め、補修回数とライフサイクルCO2を大幅に削減する実用的な材料革新を、原理（マイクロカプセル、毛細管作用、重合反応）から実証例までわかりやすく解説する。
- Value: 維持管理コストと資源消費を同時に下げ、インフラ老朽化や災害リスクに備える現実解。限られた公共予算でより安全・長寿命な都市を実現できる。
- Audience: インフラ・建築に関心のある一般視聴者、土木・材料系の学生、自治体の維持管理担当者、リフォームやDIYに関心のある人
- Education: ひび割れの力学、自己治癒機構（マイクロカプセル・血管型）、重合化学、耐久性評価、ライフサイクルアセスメント（LCA）の基礎
- Reason: 過去の“量子ドット太陽光インク”や“量子時計の重力地図”とは分野も現象も異なり、エネルギー発電や量子計測ではなく、構造材料の自己修復という新しい学びを提供する。過去候補のバイオ素材（藻類・バクテリアセルロース）とも異なり、無生物系のマイクロカプセル化学を中心に扱う。

### 2. CO2を“岩に変える”：玄武岩鉱化で数千年スケールのカーボンロック

- Score: 94
- Summary: 二酸化炭素を水に溶かして玄武岩層に注入し、カルシウム・マグネシウムと反応して炭酸塩鉱物として固定する“鉱物化貯留”。炭素-ケイ酸塩サイクルの加速、反応速度、必要な水量やトレーサーによる監視、安全性とスケール化の課題、鉱滓やコンクリートの副反応活用まで、地球化学の視点で解説する。
- Value: 再エネ導入だけでは残る排出を、長期に安定固定できる有力な選択肢。地質資源の多い地域で雇用と技術基盤を生み、実装と監視の標準化にも資する。
- Audience: 気候変動に関心のある市民、政策担当者、地質・化学・環境工学を学ぶ学生、サステナビリティ担当者
- Education: 炭酸塩形成反応、溶解・拡散・孔隙率といった地層内輸送、pHと反応速度の関係、モニタリング（アイソトープ・電気比抵抗）と安全設計
- Reason: 過去の“印刷型太陽電池”のようなエネルギー変換ではなく、地球化学による炭素固定の物質変換が主役。量子計測や通信インフラを用いた地球観測とも無関係で、全く異なる学び（反応化学・地質工学）を提供する。

### 3. 森を“立体で量る”宇宙レーザー：衛星LiDARが描く世界の炭素地図

- Score: 91
- Summary: 衛星からレーザーを打ち、樹冠から地面までの往復時間を測って高さ分布を取得する衛星LiDAR。フォトンカウント方式や波形解析で森林バイオマスを推定し、伐採や復元の変化をグローバルに追跡する仕組みと限界、レーダー・地上計測との統合、オープンデータの活用方法を紹介する。
- Value: 透明性の高い炭素会計や保全政策、復元プロジェクトの効果検証を後押し。市民もオープンデータで地域の森の変化を“見える化”できる。
- Audience: 自然や地図が好きな一般視聴者、森林・環境分野の学生、ESG・カーボンクレジットに関わる実務者、自治体の環境担当
- Education: LiDARのタイムオブフライト原理、サンプリング密度と不確実性、波形とバイオマスの関係、リモートセンシングのデータ融合
- Reason: 同じ“地図”でも、過去の“量子時計で重力を見る”は相対論的時間計測。本テーマはレーザーリモートセンシングで植生の立体構造を測るもので、インフラセンシング（海底ケーブルDAS）とも技術・対象が全く異なる。

### 4. 空中に“触れる”ボタン：超音波でつくるミッドエア・ハプティクスの世界

- Score: 95
- Summary: 多数の超音波素子で位相を制御し、空間の一点に音圧の焦点を作って“触感”を生み出す技術。非接触でクリック感やテクスチャを提示でき、医療・自動車・公共端末・VR/ARでの直感的インターフェースを実現する。仕組み（ビームフォーミング、干渉、知覚閾）、安全性、応用例を映像的に解説。
- Value: 衛生的でアクセシブルなUIを提供し、手袋や距離が必要な現場でも高い操作性を確保。触覚提示により誤操作を減らし安全性を高める。
- Audience: UI/UXデザイナー、VR/ARに関心のある人、医療・製造現場の技術者、教育関係者
- Education: 超音波の波長と焦点サイズ、位相制御とアレイ工学、皮膚の機械受容器と知覚、音圧と安全基準の基礎
- Reason: 過去候補の“音を曲げる壁（音響メタマテリアル）”は騒音制御が目的。本テーマは同じ“音”でも、メタマテリアルではなくアレイの位相制御で“触感”を生成するUI技術で、目的・原理・学びが明確に異なる。

### 5. 飲み込める“電池なし”センサー：胃酸やブドウ糖で動くマイクロ医療デバイス

- Score: 93
- Summary: 胃酸を電解質とする小型ガルバニ電池や、酵素でブドウ糖を電気に変えるバイオ燃料電池を用いて、体内で自律的に発電する経口センサー。pH・体温・出血・感染兆候などを計測し、無線で体外へ送信する仕組みと材料の生体適合性、分解性、通信の工夫を紹介する。
- Value: 侵襲を抑えた連続モニタリングで早期発見と在宅医療を支援。使い捨て電池を減らしつつ、医療アクセスが限られた地域でも検査を可能にする。
- Audience: 医療に関心のある一般視聴者、患者と家族、医療・看護・工学系の学生、メドテック開発者
- Education: 電気化学（酸化還元・起電力）、バイオ燃料電池、RF通信の減衰とアンテナ設計、生体適合材料と安全性評価
- Reason: 過去候補の“Wi‑Fiや音で壁越しに画像化”とは異なり、体内のエネルギーハーベスティングとマイクロセンサーの設計が主題。DNAデータ保存や藻類素材などの生体テーマとも重ならず、医療デバイス工学の新しい学びを提供する。
