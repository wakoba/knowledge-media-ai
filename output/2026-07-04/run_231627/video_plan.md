# Atlas Video Plan

Date: 2026-07-04

## Title

空気から飲み水をつくる？MOFと塩ゲルがひらく大気集水の最前線

## Video Concept

乾いた砂漠の朝から“1日の物語”として、大気中の水蒸気が材料で捕まり、太陽などの低品位熱で放され、冷たい面で水に戻るまでを追う。S字の吸着等温線や受動型・能動型のサイクルを、スライダーや図解・シンプルなアニメでやさしく可視化し、最後に水質・耐久・コストの現実と“気候適応型”の発想を提示する。

## Estimated Duration

10 minutes

## Visual Style

落ち着いたパステルと高コントラストのラインで描く教育向けモーショングラフィックス。粒子（●）で水分子、格子でMOF、青→赤の温度グラデーションで熱挙動を表現。アイコンは最小限・シンプル。図解・抽象アニメ中心、実写風は抽象化したジェネリックBロールに限定。

## Target Viewer

環境技術や新素材に関心のある一般視聴者、災害対策やオフグリッドな暮らしに興味がある人、高校〜大学初年級の理系学習者

## Opening Visual

砂漠の夜明け、透明な小箱の中で微細な水滴が一粒生まれ、朝日に照らされてきらめく。その背後に“空気→水”の矢印がふわりと現れる。

### Scene 2: イントロ：AWHとは

Narration Summary:

大気集水（AWH）は“加湿器の逆”。MOFや塩系ゲルが低湿でも水分子をつかみ、日中の太陽や低温の廃熱で放し、冷たい面で凝縮して回収する流れ。飲用は水質管理が前提であることも明示。

Visual Description:

空気中の点（H2O）→吸着材の層→加熱で蒸気として離脱→冷たい面で水滴→タンクのフローチャート。右下に衛生管理アイコン（凝縮面経由・ろ過・UV・ミネラル・検査）。

On-screen Text:

大気集水＝“加湿器の逆”・吸着→放湿→凝縮→回収・飲用には水質管理が前提

Asset Type:

図解アニメーション

Editing Notes:

用語に初出ルビ風ポップアップ（AWH）。重要語は1秒程度ハイライト。次セクションへスライドトランジション。

Safety Notes:

飲用は凝縮面経由での回収・ろ過・UV等の衛生処理・必要に応じたミネラル添加・地域基準に沿った検査が前提である旨を明記。

### Scene 3: 背景：なぜ“空気から水”？

Narration Summary:

災害時や遠隔地・沿岸の塩害地域では配管や電力に頼れない場面がある。どこにでもある空気から必要なときに少量でも水を得る発想と、低品位熱の活用、材料進歩で現実味が増している背景を示す。

Visual Description:

世界地図に湿度・温度のヒートレイヤー。配管や電網が脆弱・未整備な地域を点描で示す。供給手段のアイコン比較（配管・タンカー・海水淡水化・AWH）で長短を簡潔に表示。低品位熱のアイコン（太陽・廃熱・放射冷却）。

On-screen Text:

どこでもある“空気”×低品位熱・届けにくい場所で価値

### Scene 4: 分子がスイッチ：S字の吸着等温線

Narration Summary:

湿度と貯め込む水の量の関係＝吸着等温線。MOFの一部はS字になり、ある相対湿度のしきい値を越えると一気に満ちる“協同的吸着”を示す。10〜30%RHで段差が現れる設計や、50〜90°Cで放湿しやすいことを解説。

Visual Description:

湿度スライダーを動かすとS字カーブの段差を越える瞬間に、MOFの孔モデルが一気に青い水分子で満たされる。温度を上げると色が赤寄りになり、水が放出されるアニメ。

On-screen Text:

S字の吸着等温線・作動湿度（段差の位置）・作動容量（段差の高さ）・放湿目安：50〜90°C

Asset Type:

インタラクティブ風アニメ

Editing Notes:

段差通過時に“カチッ”と軽いクリック音。グラフと孔モデルを2画面で同期表示。

### Scene 5: 主役の材料：MOFと塩系ゲル

Narration Summary:

MOFは金属と有機配位子の骨組みで孔の性質を設計でき、低湿でスイッチが入るよう調整可能。低湿に強いMOF-801やMOF-303の報告がある。塩系ゲルはLiClやCaCl2などの潮解性塩を骨格に保持し、DRHを越えると強く水を引き込む。高吸水だが漏液・腐食・再生時の熱管理に配慮が必要。

Visual Description:

MOF格子の簡易CGが水分子を優先的に取り込む様子。塩ビーズが湿度上昇で液化（潮解）→ゲルに固定して漏れにくくする断面図。MOF-801・MOF-303の名称ラベルと低湿で段差が出るイメージ。

On-screen Text:

MOF：分子骨組みで“水を選びやすく”・塩ゲル：DRH越えで強吸水・LiCl目安約11%RH／CaCl2約30%RH（25°C付近）

Asset Type:

CG＋断面図アニメ

Editing Notes:

塩の液化はタイムラプス風。注意アイコンで“漏液・腐食・熱管理に配慮”を短く提示。

Safety Notes:

実験・実機の化学薬品の取り扱いは行わず、図解のみで説明。視聴者に化学薬品の無許可使用を促さない。

### Scene 6: 装置は1日のリズム：吸着→放湿→凝縮→再生

Narration Summary:

夜間〜朝に吸着、日中に低品位熱で放湿、冷たい面で凝縮・回収という基本サイクル。受動型は自然の揺らぎで1〜数サイクル/日、能動型は送風・熱交換・ヒートポンプで最適化と熱回収。断熱や薄膜化・流路工夫が鍵。

Visual Description:

朝・昼・夜のタイムライン上で“吸着→放湿→凝縮→再生”を周回。受動型の断面図（上部透明カバーで温まる吸着材と冷たい凝縮面の水滴）。能動型のフローダイアグラム（送風・熱交換・2床交互）。

On-screen Text:

受動型：自然の揺らぎで駆動・能動型：送風/熱交換/熱回収・設計ポイント：断熱・薄く塗る・流路工夫

Asset Type:

フロー図アニメーション

Editing Notes:

太陽→月のピクトを回転トランジションに活用。能動型は矢印を点滅させ熱回収ループを強調。

### Scene 7: どれくらい採れる？“条件依存”の現実

Narration Summary:

砂漠並みの乾燥でも動くMOFの報告や屋外回収実証がある一方、量は気象と装置しだい。材料・湿度・温度・日射・風・設計で大きく変わり、低湿条件でも1 kgあたり日量で数百mL〜数Lという幅の文献例がある。気候に合わせ段差位置や移動設計を最適化する重要性を伝える。

Visual Description:

世界地図に“低湿は吸着材型が有望／高湿は冷却凝縮型が有利”の色分け。天気（RH・温度・日射・風）に伴う日々の回収量の折れ線（範囲のみ）。2017年頃の屋外実証→その後の改良の年表風表示。注意テロップを下部に固定表示。

On-screen Text:

条件依存・例：低湿で1 kg当たり“数百mL〜数L/日”（文献範囲）・最適化＝気候×材料×設計

Asset Type:

情報図解＋年表アニメ

Editing Notes:

具体数値は範囲表示のみで断定を避ける。注意テロップを白フチ付きで常時可読に。

### Scene 8: 方式のちがい：冷やす？吸う？

Visual Description:

二分チャートで“冷却凝縮型”と“吸着型”の得意領域を矢印で示す。夜の屋外で放射冷却プレートが周囲より冷えていく熱画像風の可視化。砂漠＝吸着材、熱帯沿岸＝冷却凝縮などのアイコン対応図。

On-screen Text:

### Scene 9: 水は本当に飲める？安全と誤解整理

Narration Summary:

“どこでも同量”や“太陽だけで無限”ではない点を整理。MOFの安全性は系に依存し、凝縮面経由での回収と水質検査・前処理・必要に応じたミネラル添加等の管理が前提。塩ゲルは蒸気のみ通す隔離構造なら塩水混入を避け得るが、設計不良だと交差汚染の懸念が残る。環境・基準に合わせた検査・維持管理が不可欠。

Visual Description:

誤解×と正しい理解○をカードで対比。凝縮面→フィルタ→タンク→ミネラル添加→給水口のフロー。蒸気のみ通過する隔離構造の断面スケッチ。

On-screen Text:

“条件で変動する”が前提・回収は凝縮面経由・検査/衛生管理が不可欠

Asset Type:

Safety Notes:

### Scene 10: 乗り越えるべき壁：耐久・コスト・熱回収

## Subtitle Direction

専門用語は初出時に括弧で日本語補足（例：大気集水（AWH））。1行13〜16文字程度、最大2行。重要語（作動湿度・作動容量・放射冷却など）は色ハイライト。数値や単位は読みやすい全角中点で区切り（例：50〜90°C）。誤解を招く断定を避け、“条件依存”などの注意は画面下に長め表示。背景が明るいシーンでは字幕にシャドウを付与。

## Narration Direction

落ち着いたトーンで好奇心を誘う語り。ややゆっくりめ（情報密度が上がる場面はポーズを挿入）。比喩は短く具体、誇張表現は避ける。数値登場時は一拍置いて視聴者に理解の余裕を与える。最後は問いかけで余韻を残す。

## Thumbnail Direction

選択肢1：砂漠の地平線＋透明な箱の中で生まれる水滴の接写、テキスト「空気→水」。選択肢2：S字カーブの上にポンと現れる水滴、テキスト「湿度がカチッとスイッチ」。選択肢3：太陽と月のアイコンに“吸着・放湿・凝縮”が周回、テキスト「1日のリズムで水をつくる」。選択肢4：MOF格子が水分子だけをキャッチ、テキスト「分子が水を選ぶ」。どれも明瞭な2色構成＋太字サンセリフ、日本語テキストは6〜10文字に収める。

## Production Notes

- 台本外の新規データ・数値は追加しない。数値は“目安・範囲”として表示し断定を避ける。
- 実写素材は使わず、ジェネリックなアニメ・図解・抽象表現を優先。ブランドやロゴ、論文誌の実素材は不使用（文字情報の参照は可）。
- 危険・不衛生な実演は行わない。シリカゲル体験はイラスト調で“飲用不可・無着色・子ども/ペット注意・処分”の注意を常時表示。
- 水の飲用に関する描写は“凝縮面経由で回収＋ろ過/UV等の衛生処理＋地域基準の検査が前提”を必ず添える。
- 色設計はカラーブラインドフレンドリー（赤緑の同時使用を避け、青橙対比を基本）。温度は青→赤の一方向グラデ、湿度は薄青→濃青で統一。
- MOFや塩の具体的構造は簡略化して描写し、誤認を避けるために“模式図”のラベルを付す。
- BGMは静かなアンビエント。S字“段差”の瞬間に軽いクリック音、水滴形成に柔らかいベル音で理解を補助。
- 字幕・UI・グラフは全て同一フォントファミリで統一。数式表現は避け、言葉と図で直感的に伝える。
- 受動型/能動型の比較では優劣の断定を避け、“気候適応型”の観点を文言と色で強調。
- 年表・地図などの図像は著作権非依存のオリジナルで作成し、写真は使用しない。
