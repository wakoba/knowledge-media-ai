# Editorial Meeting

Date: 2026-07-04

## Topics

### 1. 厚みゼロの“レンズ”：メタサーフェスが変える光学設計

- Score: 95
- Summary: ナノスケールの柱やスリットを平面上に並べ、光の位相をピクセルごとに操って焦点や収差補正を実現する超薄型レンズの仕組みと応用を解説する。
- Value: カメラ・ARグラス・顕微鏡の小型化や新機能につながり、実写とシミュレーションで光の波面が“見える”体験を提供できる。
- Audience: 写真・カメラ愛好家、ハードウェア開発者、光学・物理を学ぶ学生、XRに関心のある人。
- Education: 位相制御、回折と波面整形、数値孔径と解像度、色収差補正の課題、製造プロセスの制約。
- Reason: 過去候補の「構造色ディスプレイ」は色表示が主題だが、本テーマは焦点形成と収差補正という光学設計の根幹に迫る。採用済みの材料・生体・音響系テーマとも重ならず、実験的ビジュアルと理論の両輪で学びが深い。

### 2. 砂漠に“網で雨”を降らせる：霧集水メッシュの設計学

- Score: 94
- Summary: 海霧を微細メッシュで捕まえ、濡れ性パターンと形状で滴を効率よく回収するフォグハーベスティングの最新研究と実装例を紹介する。
- Value: 電力を使わない給水という実用性と、DIYスケールで再現できる実験性があり、地域課題の解決に直結する希望を伝えられる。
- Audience: 水資源や環境に関心のある人、メーカーエンジニア、教育者、アウトドア愛好家。
- Education: ヌクレオーションと滴成長、接触角と接触角ヒステリシス、ラプラス圧、表面パターン設計と最適化。
- Reason: 放射冷却フィルム候補は熱放射による受動冷却だが、本テーマは湿潤空気からの受動集水で、対象現象と設計指標がまったく異なる。既採用の土木・材料テーマとも重複しない。

### 3. 海に降る“雪”が地球を冷やす？マリンスノーと炭素ポンプ

- Score: 93
- Summary: 表層で固定された炭素が有機粒子として“雪”のように沈み、深海へ輸送される生物炭素ポンプの物理と生態の相互作用を追う。
- Value: 気候と海のつながりを直感的に理解でき、沈降実験や可視化で教育コンテンツとしても強い。
- Audience: 気候変動に関心のある一般視聴者、海洋好きの子ども、理科教員、環境政策に関わる人。
- Education: ストークスの法則と沈降速度、粒子フラックス測定、炭素固定とレミネラリゼーション、時空間スケールの考え方。
- Reason: 過去候補の「内部波・海洋混合」は物理混合が主題だが、本テーマは生物地球化学サイクルに焦点を当てており、学びの軸が明確に異なる。

### 4. 地震波を“曲げる”地盤メタマテリアル：都市を守る波動工学

- Score: 96
- Summary: 地中に周期構造や空孔列を配置して表面波のバンドギャップや屈折を制御し、揺れのエネルギーを回避・散逸させる新しい耐震アプローチを紹介する。
- Value: 都市防災に直結する応用性と、卓上実験でバンドギャップを体験できる視覚教材としての強さがある。
- Audience: 土木・都市計画に関心のある人、地震に関心のある一般視聴者、理工系学生、行政担当者。
- Education: レイリー波と表面波、ブロッホ波とバンド構造、散乱と回折、スケール効果と相似則。
- Reason: 採用済みの「音響浮揚」は空中音波の操作だが、本テーマは地盤中の地震波を対象とし、メタマテリアルによるバンド構造設計が核である。Wi-Fiトモグラフィ候補の電磁波イメージングとも問題設定が異なる。

### 5. 紙一枚が“巨大構造”に：折り紙工学が拓く宇宙と建築

- Score: 97
- Summary: ミウラ折りや水準折りなどのパターンで、薄いシートを小さく収納し素早く展開する機構を設計する折り紙工学の原理と応用を解説する。
- Value: 紙で手を動かして理解でき、宇宙アンテナや建築の展開構造、災害時シェルターなど実社会の課題に直結する。
- Audience: 宇宙・建築好きの一般視聴者、デザイナー、教育者、子どもから大人まで。
- Education: 山折り・谷折りと拘束条件、自由度解析、可展面と折畳み幾何、コンプライアント機構と材料選定。
- Reason: 候補の「4Dプリンティング」は刺激応答材料で形を変えるが、本テーマは幾何と機構設計で可展性を実現するため学びの軸が異なる。採用済みの材料・生体・音響系テーマとも重ならず、家庭で再現できる体験価値が高い。
